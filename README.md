# Hedgehog RP Map
> **Основной проект:** [hedgehog-rp](https://github.com/MaSStiK/hedgehog-rp)

**Hedgehog RP Map** - интерактивная карта мира Hedgehog RP, разработанная на **React**.  
Сайт позволяет просматривать актуальную версию мировой карты проекта, а также архив всех версий карты из предыдущих сезонов.  

Карта использовалась для отображения государств участников, изменений территорий, политической ситуации и развития мира внутри ролевой вселенной Hedgehog RP.

## 🔗 Связанные проекты
- 🌍 [Hedgehog RP](https://github.com/MaSStiK/hedgehog-rp)
- 📺 [Ежиное телевидение](https://github.com/MaSStiK/tv.hedgehog-rp)
- 📊 [Статистика беседы](https://github.com/MaSStiK/stats.hedgehog-rp)

## 🛠️ Технологии
- **React** — интерфейс приложения
- **react-dropdown-select** — кастомные выпадающие списки
- **react-zoom-pan-pinch** — масштабирование и перемещение карты
